# Frontend (Next.js App Router) - Starter

Minimal scaffold for Next.js 14 App Router + TypeScript.

## Quickstart

This is a file scaffold only. To initialize a real Next.js app:

```
npm create next-app@latest frontend
# Choose App Router + TypeScript, then copy over app/ and components/ patterns below
```

## Layout
- app/layout.tsx (root layout)
- app/page.tsx (home)
- components/Nav.tsx (nav placeholder)
- styles/globals.css (optional Tailwind hook)
