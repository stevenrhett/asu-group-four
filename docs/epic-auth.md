# Epic: Authentication & Roles

## Goal
Provide secure login/registration with role-aware experiences for seekers and employers.

## Scope
- Registration, login, logout with JWT
- Password hashing (bcrypt), validation, basic rate limiting
- Roles: seeker, employer (admin optional later)

## Stories
- [ ] ST-001 Auth & JWT (seekers/employers)
- [ ] ST-001a Role-Aware Navigation (seeker/employer)
